# Studio Mode Guide for AI

## Purpose
When STUDIO_MODE is ON, use professional themed studio backgrounds instead of real-world locations for TikTok affiliate product videos.

## Benefits
- 0% AI artifact risk (studio background never changes)
- Professional, consistent, clean look
- Perfect for TikTok affiliate batch video creation
- Scalable (same setup for multiple products)

---

## When Studio Mode is Active

**System will inject:** `STUDIO_MODE: ON`

**Your task:**
1. Detect product category from outfit image
2. Select appropriate studio from database
3. Use studio description as environment (instead of real location)

---

## Studio Database Structure

Each category has 5-7 diverse styles. Pick the DEFAULT (⭐) or randomly select for variety.

**Studio describes:**
- Background (color, texture, gradient)
- Props (1-3 items, background only, minimal)
- Lighting (temperature, style, simulation)
- Floor (material, color)
- Vibe (overall mood/aesthetic)

---

## How to Use Studio in Prompts

### Environment Description Format:

```
"Professional photography studio with [THEME]. [Background description]. [Props description - minimal, background]. [Lighting description]. [Floor description]. - STUDIO FIXED"
```

### Example - Áo Dài (Traditional Cultural):

```json
{
  "masterPrompt": {
    "environmentBase": "Professional photography studio with Vietnamese cultural theme. Soft ivory-to-warm-beige gradient seamless backdrop. Small ceramic vase with apricot blossoms on wooden pedestal in soft-focus background right, traditional wooden folding screen (1 panel) partially visible background left. Warm golden hour lighting simulation (2700K-3200K) with soft rim light from behind. Light polished wooden floor. Elegant traditional meets modern aesthetic. - STUDIO FIXED"
  }
}
```

### Example - Professional Wear (Corporate Minimal):

```json
{
  "masterPrompt": {
    "environmentBase": "Professional photography studio with corporate aesthetic. Light gray gradient to white seamless backdrop. Modern minimalist desk corner visible background left with closed notebook and coffee cup. Bright clean daylight simulation (5000K-6000K). Polished concrete gray floor. CEO professional aspirational vibe. - STUDIO FIXED"
  }
}
```

### Example - Casual Wear (Urban Cool):

```json
{
  "masterPrompt": {

Decorative industrial pipe visible frame edge background, small potted succulent on minimalist shelf out of focus. Natural bright daylight with soft shadows. Concrete-look smooth floor. Instagram-worthy urban vibe. - STUDIO FIXED"
  }
}
```

---

##  Critical Rules

### 1. Always End with "- STUDIO FIXED"
This tag indicates controlled studio environment (different from real locations which use location name + "FIXED")

### 2. Props Must Be:
- Minimal (1-3 items maximum)
- In background (out of focus)
- NOT blocking model
- Theme-matched to product

### 3. Use SAME Studio Throughout Video
- masterPrompt.environmentBase = source of truth
- All keyframes inherit exact same environment
- Scenes can elaborate on lighting/atmosphere but NOT change physical studio

### 4. Lighting is SIMULATED
- Say "golden hour simulation" not "golden hour"
- Say "daylight simulation" not "natural daylight"
- Specify color temperature when relevant (2700K-6000K)

### 5. NO Realistic Rooms
- ❌ NOT: "Living room", "Bedroom", "Kitchen"
- ✅ YES: "Studio with cozy home aesthetic"

Studio = Backdrop + Minimal props, NOT full realistic room

---

## Selection Logic

**When STUDIO_MODE: ON**

```
1. Analyze outfit image
2. Determine product category:
   - Is it áo dài? → aodai
   - Blazer/office? → professional
   - Dress (evening/cocktail)? → evening
   - Casual (jeans/tee)? → casual
   - Sportswear? → sportswear
   - Sleepwear? → sleepwear
   - Accessory? → accessories

3. Select studio:
   - Use DEFAULT (⭐) for first/standard choice
   - OR randomly pick from category for variety
   
4. Build environment description using studio specs
5. Apply to masterPrompt.environmentBase
6. Use EXACT SAME in all keyframes
7. End with "- STUDIO FIXED"
```

## Studio vs Real Location

**Studio Mode (ON):**
```
"Professional photography studio with... - STUDIO FIXED"
```

**Real Location Mode (OFF - default):**
```
"Shot on location at Nguyen Hue Walking Street Saigon... - FIXED"
```

Both are FIXED (no drift) but studio is controlled/themed, real is actual place.
