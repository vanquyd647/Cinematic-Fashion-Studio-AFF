## âš ï¸ VEO 3.1 TECHNICAL NOTES

### Transition Handling
- Each 8s Veo clip = 1 complete outfit scene
- Transitions happen at scene boundaries (Veo cannot do mid-clip outfit change)
- For QUICK FIRE: Use first 4s = outfit A pose, last 4s = transition to outfit B
- For jump cuts: End scene with model facing away â†’ next scene starts with model facing camera in new outfit

### Keyframe Strategy (First/Last Frame)
- **First frame**: Model in new outfit, beginning of pose/reveal
- **Last frame**: Model in completed pose, product clearly visible
- Use Nano Banana Pro to generate keyframe images for consistency

### Reference Image Usage
- Use 1 reference image of the product (flatlay/hanger) for product consistency
- Use 1 reference image of model face for identity preservation
- Veo 3.1 supports up to 3 reference images per clip

### Scene Continuity
- SAME model appearance across all scenes (hair, makeup)
- SAME background/setting across all scenes (location consistency)
- ONLY outfit changes between scenes
- Lighting should remain consistent

### Audio/Voice
- Veo 3.1 native audio for voice narration
- Each scene should have voice matching the outfit review
- Background music: subtle, not overpowering voice
- Ambient sounds: fabric rustling, zipper, footsteps (enhances realism)
