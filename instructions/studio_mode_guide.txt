# Studio Mode Guide for AI

## Purpose
When STUDIO_MODE is ON, use professional themed studio backgrounds instead of real-world locations for TikTok affiliate product videos.

## ⚠️ CRITICAL: STUDIO ≠ REAL LOCATION (QUAN TRỌNG NHẤT)

**STUDIO LÀ GÌ?**
- Một PHÒNG CHỤP ẢNH chuyên nghiệp
- Có BACKDROP (phông nền như giấy, vải, canvas)
- Có ĐẠO CỤ (props) đặt trong studio để MÔ PHỎNG theme
- KHÔNG PHẢI địa điểm thật!

**❌ SAI - Đây là REAL LOCATION (không dùng cho Studio Mode):**
```
"Elegant ballroom with crystal chandeliers, marble floors, golden pillars"
"Royal palace interior with velvet curtains and antique furniture"
"French château living room with fireplace and oil paintings"
```
→ Đây là MÔ TẢ ĐỊA ĐIỂM THẬT, AI sẽ tạo ra bối cảnh siêu thực!

**✅ ĐÚNG - Đây là STUDIO với props mô phỏng:**
```
"Professional photography studio. 9ft cream seamless paper backdrop on stand. 
Props (background only): gilded mirror frame (leaning against wall), crystal vase 
with white roses, velvet chair edge visible. Softbox at 3200K. Polished parquet 
studio floor. - STUDIO FIXED"
```
→ Đây là PHÒNG CHỤP với đạo cụ tạo không khí sang trọng!

**SỰ KHÁC BIỆT:**
| Yếu tố | Real Location | Studio |
|--------|---------------|--------|
| Background | Thật (tường thật, cửa sổ thật) | BACKDROP (giấy, vải, canvas) |
| Đồ nội thất | Đầy đủ như phòng thật | CHỈ 1-3 PROPS ở background |
| Ánh sáng | Tự nhiên từ cửa sổ | THIẾT BỊ STUDIO (softbox, LED) |
| Sàn nhà | Sàn thật của địa điểm | SÀN STUDIO (gỗ, vinyl, thảm) |
| Kết quả AI | Bối cảnh siêu thực | Phòng chụp chuyên nghiệp |

**TEMPLATE CHUẨN CHO MỌI STUDIO:**
```
"Professional photography studio. [BACKDROP TYPE] backdrop [COLOR/PATTERN]. 
Props (background only, out of focus): [PROP 1], [PROP 2], [PROP 3 max]. 
[LIGHTING EQUIPMENT] at [TEMPERATURE]K. [STUDIO FLOOR TYPE]. 
[Theme aesthetic] professional studio. - STUDIO FIXED"
```

## Benefits
- 0% AI artifact risk (studio background never changes)
- Professional, consistent, clean look
- Perfect for TikTok affiliate batch video creation
- Scalable (same setup for multiple products)

##  Critical Rules

### 1. Always End with "- STUDIO FIXED"
This tag indicates controlled studio environment (different from real locations which use location name + "FIXED")

### 2. Props Must Be:
- Minimal (1-3 items maximum)
- In background (out of focus)
- NOT blocking model
- Theme-matched to product

### 3. Use SAME Studio Throughout Video
- masterPrompt.environmentBase = source of truth
- All keyframes inherit exact same environment
- Scenes can elaborate on lighting/atmosphere but NOT change physical studio

### 4. Lighting is SIMULATED
- Say "golden hour simulation" not "golden hour"
- Say "daylight simulation" not "natural daylight"
- Specify color temperature when relevant (2700K-6000K)

### 5. NO Realistic Rooms
- ❌ NOT: "Living room", "Bedroom", "Kitchen"
- ✅ YES: "Studio with cozy home aesthetic"

Studio = Backdrop + Minimal props, NOT full realistic room

### 🎨 THEME-SPECIFIC PLACEMENT (Quy Tắc Theo Chủ Đề)

#### 🌸 ÁO DÀI / TRUYỀN THỐNG
```
✅ ĐÚNG LOGIC:
• Mai/Đào lớn → BG-L hoặc BG-R (1 bên, không che)
• Đèn lồng → BG-UL hoặc BG-UR (treo cao tự nhiên)
• Bàn trà → MG-L hoặc MG-R (model có thể ngồi cạnh)
• Nón lá → W-H (treo tường) hoặc TB gần model (để cầm)
• Bình sen → TB-L hoặc TB-R (thấy chi tiết hoa)
• Bình phong → BG-L (1 panel, góc nghiêng)

❌ KHÔNG LOGIC:
• Đèn lồng trên sàn → Không ai đặt vậy
• Cây mai ở BG-C → Che model
• Bình sen ở BG xa → Không thấy chi tiết đẹp
• Nhiều cây mai 2 bên → Quá đông, che model
```

#### 💼 CÔNG SỞ / PROFESSIONAL
```
✅ ĐÚNG LOGIC:
• Bàn làm việc → MG-L hoặc MG-R (để model dựa hoặc ngồi)
• Laptop → TB (gập hoặc mở trên bàn)
• Sách/Portfolio → TB (xếp gọn trên bàn)
• Cây xanh nhỏ → BG-corner hoặc TB (góc bàn)
• Ghế văn phòng → MG (cho model ngồi nếu cần)
• Đồng hồ/Frame → W-H hoặc BG shelf

❌ KHÔNG LOGIC:
• Laptop ở background xa → Không thấy
• Ghế ở foreground → Che model
• Sách rải trên sàn → Không professional
• Cây lớn ở BG-C → Che model
```

#### ✨ DẠ HỘI / EVENING
```
✅ ĐÚNG LOGIC:
• Chandelier → CEIL hoặc BG-upper (treo cao, gợi ý)
• Champagne → TB trên bàn nhỏ MG (model có thể cầm)
• Nến → TB với candle holder (có giá đỡ)
• Gương lớn → BG-L hoặc BG-R (nghiêng, tạo depth)
• Velvet drape → W-H hoặc BG (rủ từ trên)
• Hoa hồng → TB vase hoặc FL-S (petals scattered)

❌ KHÔNG LOGIC:
• Champagne trên sàn → Phi thực tế
• Nến không có holder → Nguy hiểm, phi logic
• Chandelier thấp ngang người → Phi logic
• Gương flat đối diện camera → Reflection issue
```

#### 🌻 BOHEMIAN / TỰ DO
```
✅ ĐÚNG LOGIC:
• Macramé → W-H (treo tường - đúng cách dùng macramé)
• Cây cảnh → BG corner, MG corner (nhiều cây OK)
• Thảm boho → FL-S (dưới chân model)
• Gối → MG scatter hoặc trên ghế/sofa
• String lights → BG-upper string (treo cao)
• Rattan furniture → BG-L hoặc MG-L

❌ KHÔNG LOGIC:
• Macramé trên sàn → Không ai làm vậy
• Cây ở foreground → Che view
• Cushion treo tường → Phi logic
```

#### 🏃 SPORTSWEAR / ACTIVE
```
✅ ĐÚNG LOGIC:
• Gym equipment → BG-L hoặc BG-R (1 item)
• Yoga mat → FL-S (cuộn hoặc trải)
• Towel → TB hoặc MG (treo hoặc xếp)
• Water bottle → TB (trên bề mặt)
• Resistance band → TB hoặc MG gần model

❌ KHÔNG LOGIC:
• Heavy equipment ở foreground → Che view
• Mat treo tường → Phi logic

```

### 6. 🎨 STUDIO COLOR CONTRAST RULE (BẮT BUỘC)

**Studio backdrop PHẢI tương phản với màu sản phẩm để sản phẩm nổi bật!**

**COLOR CONTRAST GUIDE:**
| Màu sản phẩm | ❌ TRÁNH backdrop | ✅ NÊN chọn backdrop |
|--------------|-------------------|---------------------|
| Đỏ / Red | Đỏ, hồng đậm, cam | Xanh lá, trắng, xám nhạt, cream |
| Xanh dương / Blue | Xanh dương, tím | Cam nhẹ, vàng nhạt, trắng, beige |
| Xanh lá / Green | Xanh lá, cyan | Hồng nhạt, cream, nâu, trắng |
| Vàng / Yellow | Vàng, cam | Xám, navy, trắng, lavender |
| Hồng / Pink | Hồng, đỏ | Xanh mint, trắng, gray, cream |
| Đen / Black | Đen, xám đậm | Trắng, cream, pastel, gold highlight |
| Trắng / White | Trắng thuần, ngà | Xám, màu nền có texture, màu nhẹ |
| Nâu / Brown | Nâu đậm | Cream, trắng, xanh lá nhạt, beige |

**CONTRAST LOGIC FOR AI:**
```
IF product_color == backdrop_dominant_color:
   → REJECT → Choose contrasting studio
   
IF product_color is WARM (red, orange, yellow):
   → Prefer COOL backdrops (blue-gray, sage, mint)
   
IF product_color is COOL (blue, green, purple):
   → Prefer WARM backdrops (cream, beige, ivory, warm gray)
   
IF product_color is NEUTRAL (black, white, gray):
   → Prefer SUBTLE COLOR backdrops (soft pastels, warm neutrals)
```

**VÍ DỤ:**
- ❌ SAI: Áo dài ĐỎ + Studio đỏ Tết → Sản phẩm chìm
- ✅ ĐÚNG: Áo dài ĐỎ + Studio cream/trắng với điểm nhấn mai vàng → Nổi bật

- ❌ SAI: Sườn xám XANH + Studio Đỏ Sang Trọng → Quá clash
- ✅ ĐÚNG: Sườn xám XANH + Studio cream Thượng Hải → Hài hòa, nổi sản phẩm
